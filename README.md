## Description

An app to show latest headlines on business related topic in the US.

### Features

- User can see the news with titles in a list
- User can see the details of the news with full title and image


### Built with

- Android
- Kotlin
- MVVM
- Dagger-hilt
- Retrofit
- Room
- Pagination (Paging 3)
- Unit testing
- Single Activity lifecycle
- Glide
- Jetpack components
- Flow
- LiveData
